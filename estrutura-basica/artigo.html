O que é HTML?

É a linguagem que descreve a estrutura e o conteudo semantico de um documento da web.
Não é linguagem de programação, e sim marcação.
Hyper Text Markup Language.

OBS:

Ispecionar código fonte de outros sites para ver o código HTML e aprender.

Chrome e o HTML:

O Browser (Chrome), interpreta e renderiza o HTML.
O documento HTML é composto por um conjunto de elementos, juntos definem a semantica e o conteúdo.

Elementos

<> é uma tag de abertura

< / > é uma tag de encerramento

< body >
Indica o corpo do documento
< / body >

< head >
< title > Titulo da pagina < / title >
< / head >

Indicar que o Documento é HTML ( Obrigatório )
No inicio colocar:
< ! DOCTYPE html >

Ajustar caracteres especiais - Codificação
UTF - 8
< head>
< meta charset="utf-8">
< /head>

Primeiras Tags: Titulos, quebras de linhas e parágrafros
< h1> Titulo principal < /h1> 1 até o 6
< p> Paragráfo < /p>
< br> braked, quebra linha.

comentar o código HTML
< !-- COMENTARIO -->

Ênfase, importância e marcação

< strong> Importancia a palavra/frase < /strong>

< em> < /em> da enfase a palavra/frase

< mark> < /mark> da relevância para palavra/frase

Imagem
< img src = img.png> na mesma pasta, busca só com nome
< img src = img/img.png> na pasta img
< img src = img/img.png width="390" height="200"> altera tamanho

Links
< a href="www.google.com.br"> palavra com link < /a>
< a href="www.google.com.br" target="_blank"> palavra com link < /a> abre em nova aba

DIV é um container genérico que não significa nada em especifico
< div> < /div>

< footer> rodapé

< aside> espaço para conteudo relacionado tipo anuncios < /aside>
